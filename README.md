# Random java util libraries

Example pom.xml:

```
<repositories>
    <repository>
        <id>jitpack.io</id>
        <url>https://jitpack.io</url>
    </repository>
</repositories>
```

```
<dependencies>
    <dependency>
        <groupId>com.github.marsik.java-utils</groupId>
        <artifactId>bugzilla</artifactId>
        <version>master-SNAPSHOT</version>
    </dependency>
</dependencies>
```
